All development in this repository must strictly adhere to the architecture, coding styles, and instructions defined in the `/ai` directory.

Before performing any coding, refactoring, or architectural task:
1. **Scan the `/ai` folder**: Read all `.md` files within the `/ai` directory to understand the project-specific rules.
2. **Follow Instructions**: Apply the logic found in `/ai/instructions.md` (or your specific filenames) to all code generation.
3. **Respect Coding Style**: Ensure variable naming, folder structure, and design patterns match the definitions in your AI-driven documentation.
